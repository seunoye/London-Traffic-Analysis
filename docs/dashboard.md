London Borough Traffic Summary 2023 dashboard created with Looker Studio. This dashboard visualises key traffic metrics by borough, including:

- Total motor vehicles, pedal cycles, and buses/coaches
- Active monitoring points and cycle share percentage
- Top 10 boroughs by total traffic (bar chart)
- Traffic composition by vehicle type (pie chart)
- Detailed traffic summary table by borough

The dashboard enables quick insights into traffic patterns and modal share across London for 2023.
